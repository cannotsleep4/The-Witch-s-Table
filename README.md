# The-Witch-s-Table

# 魔女与养子 — 轻量级策划案
*概念设计文档 v0.1*

---

## 一、核心概念

一个永生的造物主魔女，在漫长的岁月中对一切感到厌倦。她创造了一面魔镜，用来给自己当DM，一遍遍在自己创造的世界里养育一个孩子.

**类型**：养成模拟 / 剧情探索  

**参考**：密教模拟器（界面与玩法）、美少女梦工厂（养成循环）、火山的女儿（双视角情感）、邪恶铭刻（周目永久变化）  

**平台**：PC

**引擎**： godot

---
## 二、界面设计
游戏开始，玩家坐在一张大桌子前，对面是一个笼罩在阴影里、只露出手和上半身的人影。桌上摆着水晶球、骰子、代表养子的木头小人棋子。

人影引导玩家选择：
- 养子的种族与外貌
- 捡到养子的地点（对应不同初始属性与乘区）

**一周目结束后**，玩家可以起身走开——这时发现对面的人影不是人影，而是一面镜子。玩家一直在和自己对话。

> *真相：魔女用魔镜扮演命运，给自己当DM。*

## 三、视角与镜头语言

| 场景 | 视角 | 含义 |
|------|------|------|
| 开场/角色创建 | 正对，看到胸膛+双手 | 平等的谈判，共同决定命运 |
| 日常养成规划 | 俯视，只见双手 | 魔女掌控日常，见证者姿态 |
| 战斗 | 稍平视，对弈感 | 与命运博弈，棋局紧张 |

## 四、世界观

### 基本设定
- 中世纪奇幻风格
- 一个走向黄昏的大帝国：老国王将死，诸子夺嫡，暗流涌动
- 魔女是这个世界的**真正造物主**，世界在她的桌面上展开
- 世界内部，魔女的身份是**隐士**，同时也是**异端**

### 四个方向的势力

| 方向 | 势力 | 定位 |
|------|------|------|
| 🌲 森林 | 古老魔法秩序（类沉睡魔咒） | 剧情与身世，魔法传承 |
| 🌋 火山 | 龙与恶魔 | 战斗副本，资源，试炼 |
| 👑 帝国中心 | 人类王权，教会 | 政治主线，养子的命运舞台 |

### 教会的秘密
教会信奉的圣母，是人们对造物主的想象。魔女认可了这个皮套，并给予真实的神力。信徒的祈祷是灵验的，奇迹是真实的——只是圣母背后是一个在桌子前玩桌游的老太太。

**隐藏剧情**：走圣职路线的养子，某个周目会发现神殿圣母雕像的脸和自己的养母一模一样。

---

## 四、出身地系统

捡弃婴的地点决定初始属性和乘区，多周目玩家可发现**特定地点+特定种族**的组合会解锁隐藏身世剧情：

- 🌲 森林边境 + 特定种族 → 可能触发精灵王寻子事件
- 🌋 火山附近 + 特定种族 → 恶魔王的契约追来了
- 🏰 战场废墟 + 人类 → 看似普通，可能是王室血脉
- 🌊 港口 + 未知种族 → 海那边来的什么人的孩子……

魔女是否告知养子真实血脉，**取决于玩家选择**。

---

## 五、核心玩法循环

### 时间系统（密教模拟器式）
- 时间读条持续推进
- 需要持续放入**金钱卡牌**维持生计
- 资源压力下做选择，永远没有最优解

### 卡牌系统
养子的状态不用数字显示，用**卡牌的数量和质量**呈现：

| 卡牌类型 | 含义 |
|----------|------|
| 💰 资源卡 | 金钱、食物、药材 |
| 📚 知识卡 | 教导技能，解锁小屋谜题 |
| 💜 情感卡 | 陪伴与对话产生，影响心理健康 |
| ⚔️ 委托卡 | 接了有收入，但占用时间 |
| ❤️ 健康卡 | 数量和质量反映身体状态 |
| 🧠 心理卡 | 反映精神状态 |

### 三条生命线
- 🍞 **饱食** — 断粮则饿死
- ❤️ **健康** — 忽视病情则病死
- 🧠 **心理** — 孤立/过度训练/接触禁忌知识则发疯致死

三条线互相影响：饿久了会生病，生病久了会发疯。

### 死亡结局的魔女反应
- **饿死**：茫然，轻描淡写的自责。*"啊……我忘了。"*
- **生病**：懊悔，带着"如果当时"的反复
- **发疯**：沉默

---

## 六、小屋系统（邪恶铭刻式）

游戏外层是魔女的小屋，可以自由探索。

**关键特性**：小屋随周目**永久变化**，每一局都留下痕迹，无法重置。展示柜越来越满，蜡烛燃短，墙上多了划痕，某本书翻到了新的一页……

### 展示柜
- 每个养子结局后，木头小人放入展示柜收藏
- 点开小人可查看这段经历的简短记录
- 也可以**销毁**小人——放进壁炉，消失，不留痕迹

### 解谜
- 小屋里藏着世界观碎片和魔女来历的线索
- 多周目逐渐解锁，某些抽屉第一周目打不开
- **真结局的触发条件**藏在小屋里，靠玩家自己拼凑

---

## 七、结局体系

| 结局 | 触发方向 | 基调 |
|------|----------|------|
| 🗡️ 弑母结局 | 养子羽翼丰满，反杀魔女 | 魔女在镜子后鼓掌，这是她写的剧本之一 |
| 💒 母嫁结局 | 养子与魔女结合 | 她最喜欢的狗血剧情，剧本也是她写的 |
| 🌍 出走结局 | 好奇心极高，决定出海远航 | 养子与海中巨兽搏斗，至死。小木头人走到地图边缘 |
| 💀 死亡结局 | 饿死/病死/发疯 | 这一局没玩好，重来 |
| ✨ 真结局 | 多周目解锁，小屋谜题拼凑完整 | 玩家终于理解她为什么一次又一次坐回那张桌子前 |

**所有养子最终都会死。魔女见证他们的每一个结局，然后重新开始。**

---

## 十、多周目设计

- 每次新游戏，魔女再次坐到镜子前，再次去捡那个孩子
- 小屋永久变化，积累每一局的痕迹
- 隐藏身世、隐藏剧情、小屋谜题随周目解锁
- 真结局需要积累足够多的周目理解
